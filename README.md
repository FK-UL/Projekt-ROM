# Projekt-ROM
## Modne besede – najbolj priljubljene besede s področja tehnologije in računalništva skozi leta
V okviru svojega projekta bom z uporabo programa Mathematica in vgrajenih funkcij jezika Wolfram analiziral različne besede, ki so se uporabljale v povezavi z novimi tehnologijami in računalniki, ter ugotovil, kdaj so bile priljubljene, kako dolgo so se uporabljale itd.

##Načrt:
- Zbral bom podatke iz različnih virov (verjetno pa bom citiral tiste, ki so že na voljo)
- Pregledal bom članke in prispevke ter izbral besede za testiranje (tj. analizo)
- Analiziral bom podatke in ustvaril ustrezne grafe, pri čemer bom izpostavil zanimive stvari
- Nekaj besed o trenutno najbolj priljubljenih buzzwordih in prihodnosti
